QUICK START COMMANDS:
=====================

1. Deploy Backend to Railway:
   → Visit https://railway.app/
   → Connect GitHub repository
   → Add PostgreSQL and Redis
   → Set environment variables
   → Deploy automatically (2-3 minutes)

2. Deploy Frontend to Netlify:
   → Visit https://app.netlify.com/
   → Connect GitHub repository
   → Configure: base=frontend, build=npm run build, publish=frontend/dist
   → Set VITE_API_URL and VITE_WS_URL
   → Deploy automatically (2-3 minutes)

3. Using Deployment Script:
   chmod +x scripts/deploy-to-netlify.sh
   ./scripts/deploy-to-netlify.sh --prod
